# Hospital Management System (Frontend Only)

### Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage based CRUD

### Features
- Role-based Dashboards (Admin, Doctor, Patient)
- Admin: Manage Doctors / Patients / Appointments
- Doctor: View Appointments / Prescriptions
- Patient: Register, Book / Cancel Appointments, View History

### Setup
1. Download / clone repository.
2. Open `index.html` in any browser.
3. System persists data in browser LocalStorage.
